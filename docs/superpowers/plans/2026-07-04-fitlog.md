# FitLog Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** localStorage 기반 개인용 헬스 트래커 웹앱(운동 기록·체중·통계·루틴)을 Vanilla JS로 구현한다.

**Architecture:** 빌드 도구 없는 정적 웹앱. index.html이 4개 탭 화면을 모두 포함하고, 일반 `<script>` 태그로 js 모듈 9개를 순서대로 로드한다. 각 탭은 자기 render 함수를 갖고 데이터 변경 시 해당 탭만 다시 그린다. 모든 데이터는 localStorage 키 4개에 즉시 저장된다.

**Tech Stack:** Vanilla JavaScript (ES2017 수준, ES 모듈 미사용), HTML, CSS, SVG (그래프 직접 구현), localStorage.

## Global Constraints

- 외부 라이브러리/CDN/폰트 사용 금지. `file://`로 열어도 완전 동작해야 함.
- ES 모듈(`import/export`) 사용 금지 — 전역 네임스페이스 객체(`FitLog.*`)로 모듈 간 통신.
- localStorage 키: `fitlog_sessions`, `fitlog_bodyMetrics`, `fitlog_routines`, `fitlog_exerciseDB` (이 4개만).
- 날짜 문자열 형식: `YYYY-MM-DD` (로컬 타임존 기준).
- 세션은 날짜당 1개.
- 1RM 공식: Epley `weight * (1 + reps / 30)`.
- 모바일 우선, 다크모드 기본. 터치 영역 최소 44px.
- UI 문구는 전부 한국어.
- 각 태스크 완료 시 커밋. 커밋 메시지 끝에 `Co-Authored-By: Claude Fable 5 <noreply@anthropic.com>`.
- 검증은 브라우저에서 직접 수행(테스트 프레임워크 없음). 각 태스크의 "검증" 스텝 체크리스트를 전부 통과해야 완료.

```js
window.FitLog = {};

FitLog.ui = {
  switchTab(tabId) {
    document.querySelectorAll('.tab-panel').forEach(p =>
      p.classList.toggle('active', p.id === 'tab-' + tabId));
    document.querySelectorAll('.tab-btn').forEach(b =>
      b.classList.toggle('active', b.dataset.tab === tabId));
    // 이후 태스크에서 탭별 onShow 렌더 호출 추가
  },
  toast(message) {
    const el = document.getElementById('toast');
    el.textContent = message;
    el.hidden = false;
    el.classList.add('show');
    clearTimeout(this._toastTimer);
    this._toastTimer = setTimeout(() => {
      el.classList.remove('show');
      el.hidden = true;
    }, 2000);
  }
};
```

- [ ] **Step 2: js/storage.js 작성** — 위 Interfaces의 함수 전부. 게터는 `load(key) || 기본값`(배열 키는 `[]`, exerciseDB는 `{ bodyParts: [] }`). `todayStr()`:

```js
todayStr(d) {
  const dt = d || new Date();
  const y = dt.getFullYear();
  const m = String(dt.getMonth() + 1).padStart(2, '0');
  const day = String(dt.getDate()).padStart(2, '0');
  return y + '-' + m + '-' + day;
}
```
